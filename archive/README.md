# Archive

Old or exploratory plotting scripts not used in the final manuscript.

- `plot_calibrations.py` — Early calibration diagnostic plots (superseded by `plot_figs3_calibration.py`)
- `plot_sims.py` — Single-sim diagnostic plots used during development
- `plot_extra_epi.py` — Additional syphilis epidemiology diagnostics (all syphilis prevalence by SW, ever-exposed time series, syphilis prevalence by HIV status bars). Not in manuscript but may be useful for presentations.
